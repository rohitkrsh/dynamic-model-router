# VectorSenseAI starter notes

This repository now includes a lightweight semantic-routing starter that can serve as the base for a broader VectorSenseAI project.

## Current scope
- semantic similarity-based prompt classification
- basic provider and routing metadata
- a simple FastAPI endpoint for routing requests

## Suggested next steps
- add a small example dataset for real retrieval use cases
- add a configurable embedding backend
- add a basic similarity index or vector store abstraction
- add README examples and a demo script
